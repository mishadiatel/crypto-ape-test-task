# Next.js Crypto Chart 

This project is a Next.js page displaying a chart with the price of Bitcoin or Solana.

## Features

- Preloader animation using framer-motion.
- Crypto icons from react-icons.
- Chart data is from api https://www.coingecko.com/.
- Ability to switch between Bitcoin and Solana.
- When switching cryptocurrencies, a preloader is shown to simulate data loading.
